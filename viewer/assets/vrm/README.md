# VRM Default Placement

Put your default avatar file here.

- Recommended: `viewer/assets/vrm/default.vrm`
- Also supported: `viewer/assets/vrm/default.glb` / `viewer/assets/vrm/default.gltf`

The suit dashboard auto-detects these files in this order:

1. `default.vrm`
2. `default.glb`
3. `default.gltf`

Open:

- `http://localhost:8010/viewer/suit-dashboard/`
